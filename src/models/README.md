# Models

Esta pasta contém as entidades do domínio representadas com modelos Pydantic.

## Objetivo

Os models definem o formato dos dados usados em toda a aplicação.

## Arquivos

- `domain.py`
  - Classe base compartilhada por todas as entidades do domínio.

- `registration.py`
  - Entidade de inscrição.

- `event.py`
  - Entidade de evento.

- `speaker.py`
  - Entidade de palestrante.

- `activity.py`
  - Entidade de atividade.

- `checkin.py`
  - Entidade de check-in.
