# Core

Esta pasta contém utilitários centrais compartilhados pela aplicação.

## Objetivo

`core/` concentra configurações e helpers reutilizáveis, sem regras de negócio.

## Arquivos

- `config.py`
  - Lê o `.env` e centraliza as configurações da aplicação.

- `uploads.py`
  - Encapsula helpers de upload, nomes de arquivos e validações de imagem.
