# Usecases

Esta pasta contém a lógica de negócio da aplicação.

## Objetivo

Os use cases orquestram o fluxo entre rotas, repositórios e regras de validação.

## Exemplos comuns

- criar evento
- criar inscrição
- realizar check-in
- exportar relatório CSV
- exportar relatório ZIP
- calcular hashes
- executar vacuum em versões antigas do Delta Lake
